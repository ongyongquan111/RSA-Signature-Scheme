RSA Digital Signature Scheme (Java GUI)

Group Members
1. Ong Yong Quan
2. Goh Xin Ran
3. Chee Jiong King


Features
* RSA 1024-bit key generation (512-bit p, q)
* SHA-512 file hashing
* Digital signature creation using RSA private key
* Signature verification using RSA public key
* View report (digest, signature, verification)


How to Run
1. Open the project in NetBeans or any Java IDE
2. Run the `RSASignatureGUI` class
3. Use the GUI buttons to:
  * Generate Keys
  * Sign a file
  * Verify the signature
  * View report


Dependencies
* Java 8 or higher
